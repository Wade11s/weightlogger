# Weight Logger 用户指南

## 欢迎使用 Weight Logger

Weight Logger 是一款简单美观的体重追踪应用，帮助您记录和追踪体重变化，实现健康目标。

## 快速开始

### 首次使用

1. **启动应用** - 双击应用图标打开 Weight Logger
2. **设置个人资料** - 点击侧边栏的「目标」选项，设置您的身高
3. **添加体重记录** - 在首页点击「快速记录」输入您的体重
4. **查看趋势** - 切换到「统计」页面查看体重变化趋势

## 功能介绍

### 📊 概览页面

- **当前体重卡片** - 显示最新一次记录的体重
- **BMI 指数** - 根据身高和体重自动计算 BMI
- **目标进度** - 显示距离目标体重的进度
- **快速记录** - 一键添加今日体重

### 📝 记录页面

- **日历视图** - 以日历形式展示所有体重记录
- **添加记录** - 点击日历中的任意日期添加记录
- **编辑记录** - 点击已有的记录进行编辑或删除

### 📈 统计页面

- **趋势图表** - 可视化展示体重变化趋势
- **时间筛选** - 支持 7天、30天、全部数据筛选
- **统计摘要** - 显示平均体重、最大/最小体重、BMI 等

- **趋势统计** - 最高体重、最低体重、总变化量

### 🎯 目标页面

- **目标设定** - 设置目标体重和目标日期
- **个人资料** - 设置身高、性别等信息
- **进度追踪** - 查看目标达成进度和预计达成日期

### ⚙️ 设置页面

- **数据导入** - 从 JSON 或 CSV 文件导入数据
  - 支持冲突处理：覆盖/保留/跳过
  - 自动验证数据格式
- **数据导出** - 导出为 JSON 或 CSV 格式
- **备份恢复** - 创建完整备份或从备份恢复

## 数据管理

### 导入数据

1. 点击「设置」>「数据导入」
2. 选择导入格式（JSON/CSV）
3. 选择冲突处理方式：
   - **覆盖** - 用导入的数据替换同日期记录
   - **保留** - 同时保留导入和现有的记录
   - **跳过** - 不导入有日期冲突的记录
4. 选择文件并确认

### 导出数据

1. 点击「设置」>「数据导出」
2. 选择导出格式（JSON/CSV）
3. 选择保存位置
4. 确认导出

### CSV 格式说明

如果您想从其他应用导入数据，请确保 CSV 文件格式如下：

```csv
Date,Weight,Note
2024-01-15,70.5,早上空腹
2024-01-16,70.2,
2024-01-17,70.0,运动后
```

**格式要求：**
- Date: YYYY-MM-DD 格式
- Weight: 数字，单位 kg，范围 20-300
- Note: 可选，备注信息

### JSON 格式说明

导出的 JSON 文件格式如下：

```json
[
  {
    "id": "unique-id",
    "date": "2024-01-15",
    "weight": 70.5,
    "note": "备注内容",
    "created_at": "2024-01-15T10:00:00Z"
  }
]
```

## 健康指标

### BMI 计算

应用会根据您的身高和体重自动计算 BMI：

| 分类 | BMI 范围 |
|------|----------|
| 偏瘦 | < 18.5 |
| 正常 | 18.5 - 23.9 |
| 超重 | 24.0 - 27.9 |
| 肥胖 | ≥ 28.0 |

## 主题切换

应用支持浅色/深色主题切换：
- 点击右上角的 🌙/☀️ 图标即可切换主题
- 主题设置会自动保存

## 数据存储

- 所有数据保存在本地：`~/.weightlogger/data.json`
- 建议定期创建备份
- 数据不会上传到任何服务器

## 快捷键

| 功能 | 快捷键 |
|------|--------|
| 保存记录 | Enter |
| 取消编辑 | Escape |

## 常见问题

**Q: 如何删除一条记录？**
A: 在记录页面点击要删除的记录，然后点击「删除」按钮。

**Q: 导入数据时日期冲突怎么办？**
A: 在导入前选择冲突处理方式：覆盖现有记录、保留两条记录、或跳过冲突记录。

**Q: 数据会同步到云端吗？**
A: 不会，所有数据仅保存在本地。请定期创建备份以防数据丢失。

**Q: 如何修改目标体重？**
A: 进入「目标」页面，修改目标体重后保存。

**Q: 支持哪些平台？**
A: 目前支持 macOS 和 Windows。
